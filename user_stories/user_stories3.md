User Stories for FoodPandas App
Account Creation - Priority: High - Estimate: 1 day
As a new customer, I want to create an account using my email and password so that I can place orders and track my delivery history.

Account Login - Priority: High - Estimate: 1 day
As a returning customer, I want to log in using my saved credentials so that I can quickly access my account.

Browse by Cuisine - Priority: High - Estimate: 3 days
As a customer, I want to browse restaurants by cuisine type so that I can find food that matches my preferences.

Search Restaurants/Dishes - Priority: Medium - Estimate: 3 days
As a customer, I want to search for specific restaurants or dishes by name so that I can quickly find what I'm craving.

View Restaurant Menu - Priority: High - Estimate: 5 days
As a customer, I want to view a restaurant's full menu with photos so that I can see what's available before ordering.

Add Items to Cart - Priority: High - Estimate: 5 days
As a customer, I want to add items to my cart so that I can order multiple dishes at once.

Card Payment - Priority: High - Estimate: 8 days
As a customer, I want to pay using a credit or debit card so that I can complete my purchase securely.

Order Confirmation - Priority: Medium - Estimate: 2 days
As a customer, I want to receive an order confirmation with details so that I have a record of my purchase.

Real-time Order Tracking - Priority: Medium - Estimate: 5 days
As a customer, I want to track my order status in real-time so that I know when my food will arrive.

Status Notifications - Priority: Medium - Estimate: 3 days
As a customer, I want to receive notifications when my order status changes so that I stay informed without checking the app constantly.
